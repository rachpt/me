Linux mint 安装 Materials Studio 8.0

发表于 2018-01-11 | 更新于: 2018-03-05 | 分类于 教程 | | 热度: ℃
字数统计: 183 | 阅读时长 ≈ 1

linux mint 是基于ubuntu的，所以对ubuntu也应该是有效的。（推荐 desktop 使用 mint cinnamon 版）

Brief introduce of MS’s setup on ubuntu linux.

author rachpt

 

setp1. 安装 rpm2cpio

step2. 使用 bash csh

step3. sudo ./install /opt/ms80

step4. cd /opt/ms80/LicensePack/etc

step5. sudo ./lp_config

step6. sudo ./lp_echovars

step7. sudo ./install /opt/ms80 y

OK! It work’s!

 

更新 2018.01.12

安装到自己的家目录（/home/user）重启后才有用，无需 sudo。

使用 bash 很重要，不能使用 ubuntu默认的 dash，否则软件无法安装，无法运行！ 具体参考这两篇博文。

http://blog.csdn.net/hemmingway/article/details/49301873

http://blog.csdn.net/gatieme/article/details/52136411

 

自己看 install.log , 设置开机启动 有个 gateway 什么的 start.

浏览器 http://127.0.0.1:18888 查看是否成功。

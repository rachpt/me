ps 命令

ps 命令的参数有：

参数 作用
-a 显示所有进程（包括其他用户的）
-u 用户以及其他信息
-x 显示没有控制终端的进程

特别地，ps aux 这种短格式不加减号（-）也是合法的。

参数 作用

-d 上面两者同时修改
